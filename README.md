# KSRO LAW ACADEMY

Web oficial de **KSRO LAW ACADEMY** — la academia líder en España sobre Inteligencia Artificial aplicada al Derecho.

**Creador:** Óscar Casero Miguel · Abogado · 28 años en tribunales
**Dominio:** [ksrolawacademy.com](https://ksrolawacademy.com)

## 🎯 Stack

- HTML5 + CSS3 + JavaScript Vanilla
- Single-file architecture (`index.html`)
- Responsive mobile-first
- SEO optimizado + Open Graph tags

## 🚀 Deploy en Vercel

Este proyecto está desplegado en Vercel con auto-deploy desde `main`.

```bash
# Deploy manual con Vercel CLI
vercel --prod
```

## 🧩 Estructura

- `index.html` — Sitio completo (estilos y JS inline)
- `styles.css` — Estilos adicionales (si aplica)
- `vercel.json` — Configuración de Vercel
